Enterprise AI Agents – Legal Data Automation

Designed and deployed production-grade AI agents to automate legal data workflows, including property tax retrieval and structured information extraction from internal systems.

Technology Stack

  • Languages & Frameworks: Python, Flask, LangChain
  • ML & AI Tools: LLMs, MLflow, Arize

Key Contributions & Impact

  • Built a LangChain-powered natural language agent that interprets user intent, dynamically selects the appropriate internal database, generates SQL via LLM reasoning, and returns auditable, structured responses.
  • Integrated secure LLMOps pipelines using MLflow and Arize to track agent behavior, query accuracy, drift, and response reliability in a governed enterprise environment.
