Predictive Modelling – IMDB Movie Review Sentiment Analysis

Built and evaluated transformer-based sentiment classification models on the IMDB 50K movie review dataset, classifying reviews into positive and negative sentiments.

Technology Stack

  • Languages & ML: Python, PyTorch, Transformers, Scikit-learn
  • Frameworks: Hugging Face

Key Contributions & Impact

  • Implemented a custom RoBERTa + Feedforward Neural Network (FFN) architecture using contextual embeddings and compared performance against Naive Bayes and RoBERTaForSequenceClassification.
  • Designed a robust text preprocessing pipeline including HTML/URL removal, contraction expansion, diacritic normalization, chat-word replacement, and punctuation handling to improve data quality.
